# 60 mm Pipe Cover

## Overview

The 60 mm Pipe Cover is a 3D-printed cover designed to fit onto 60 mm pipes. The purpose of this design is to prevent debris, small objects, and other unwanted materials from entering the pipe. The cover uses a friction-fit design, allowing it to be installed and removed without needing screws, glue, or extra hardware.

This project was designed in Onshape and printed using PETG filament to create a durable and functional part.

## Features

- Designed for 60 mm pipes
- Friction-fit installation
- One-piece design
- No screws or adhesives required
- 3D printed using PETG
- Easy to install and remove

## Design

The pipe cover was designed using Onshape CAD software. The design focuses on simplicity and ease of use by creating a one-piece part that securely fits onto a pipe using friction.

The design can be modified in CAD to create versions for different pipe sizes.
